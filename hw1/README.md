# Log Analyzer v0.1

A small python script for calculating some stats of log file.

## Installation

Python 3.x required.

## Usage

Fill config as example below:

REPORT_SIZE=1000  
REPORT_DIR=./reports  
LOG_DIR=./log  

Run:

python log_analyzer.py -c config

## Testing

python test_log_analyzer.py
